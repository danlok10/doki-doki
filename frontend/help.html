Basic Help

To advance through the game, left-click.

Game Menu

In the main screen, you can chose to start a new game, load a previous one, open the settings menu, open this help file or go to an empty page with the quit button.

When playing a game, press the Save or Load buttons in the bottom of the text screen to enter the game save screen. The save screen gives the following choices:

Return
Returns to the game.
Save Game
Allows you to save a game by clicking on a save slot.
Load Game
Allows you to load a game by clicking on a save slot.
Delete
Allows you to delete a game by clicking on a save slot.
Main Menu
Returns to the main menu, ending the current game.

Key and Mouse Bindings

Left-click
Advances through the game, activates menu choices, buttons, and sliders.
Right-click
Hides the text window and other transient displays.
